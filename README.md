# 🛒 Vendas Online

Projeto de loja virtual, primeiro  desenvolvimento de site como atividade acadêmica, com foco em estrutura de páginas web utilizando HTML e CSS.

---

## 🚀 Sobre o projeto

Este projeto simula uma loja online com navegação entre páginas, exibição de produtos e layout responsivo.

O objetivo principal é praticar:
- Estruturação de páginas web
- Organização de arquivos
- Estilização com CSS
- Navegação entre páginas

---

## ⚠️ Funcionalidades

✔ Página inicial com produtos  
✔ Navegação entre páginas  
✔ Layout visual organizado  


---

## 🛠️ Tecnologias utilizadas

- HTML5  
- CSS3
